Formato do HTML

<!DOCTYPE html>
<html>
    <head>
        tags com informações para o navegador
    </head>
    <body>
        conteúdo para o usuário
    </body>
</html>

HTML: 

<!DOCTYPE html> - Especifica qual versão do html você esta utilizando no caso 5.
<html> - Especifica que é um arquivo html.
<head> - Especifica que é o Cabeçalho do html.
<body> - Especifica o corpo da página, o que vai de fato mostrar ao usuário.   
<title> - Título da aba.
<meta charset="utf-8"> - Tipo de encoding que é especificado.
<link rel="icon" href="favicon.png"> - Cria o link com arquivos, rel é o tipo no caso icon da aba, href(referência) é o caminho do ícone. 
<link rel="stylesheet" href="bio.css"> - Especifica o link do CSS.
<h1> ou <h2> e vai crescendo até o h6- Cabeçalho.
<p> - Paragrafo.

CSS:

font-size: 60px; - Tamanho da Fonte.
background-color: rgb(red, green, blue) - Cor de fundo.
color: rgb(red, green, blue); - Cor de texto decimal.
color: #FAFFFA - Hexadecimal.
font-family: "Arial"; - Tipo de Fonte.
font-family: "Open Sans", "Arial"; - Testa as letras que o usuário tem no computador.
font-family: "Open Sans", sans-serif; - Sem aspas ele pega do navegador a fonte.
